Eid Card Maker

Simple client-side Eid greeting card maker.

Usage:

- Open `EidCard.html` in a browser.
- Upload a photo, adjust the size and message, then click `কার্ড তৈরি করুন`.
- Download or save cards locally using the provided buttons.

Production notes:

- The canvas is DPI-aware for crisp rendering on high-DPI displays.
- Add a `favicon.ico` to the project root for a site icon.
